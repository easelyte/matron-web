# Project origin

The Matron Web codebase began as a fork of Element Web. The current standalone repository keeps the copyright notices and licences required by the code that remains.

The current application was reduced to a Matron-specific matron-journal client and no longer contains or depends on the former Matrix runtime. Some long-standing DOM class names remain temporarily to keep the visual migration reviewable; they are internal implementation details, not an Element compatibility API.

Element is a project of New Vector Ltd and contributors. Matrix is an open standard stewarded by The Matrix.org Foundation. Matron is not affiliated with or endorsed by either organisation.
